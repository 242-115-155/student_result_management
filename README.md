# 🎓 Student Result Management System

A web-based **Student Result Management System** developed using **PHP, MySQL, HTML, CSS, Bootstrap, and JavaScript**. This project was developed as a **Database Management System (DBMS)** course project for the Department of Computer Science & Engineering, Metropolitan University.

---

## 📌 Features

### 👨‍💼 Admin Panel
- Admin Login
- Dashboard
- Manage Teachers (Add, Edit, Delete)
- Manage Students (Add, Edit, Delete)
- Manage Courses  (Add, Edit, Delete)
- Manage Batches  (Add, Edit, Delete)
- Manage Results  (Add, Edit, Delete)
- View Statistics

### 👨‍🏫 Teacher Panel
- Teacher Login
- Dashboard
- Add Student
- Add Result
- Edit Result
- Delete Result
- View Result

### 👨‍🎓 Student Panel
- Student Login
- View Semester Results
- View GPA / CGPA
- Search Results

## 💻 Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/student-result-management-system.git
```

2. Copy the project folder into

```
xampp/htdocs/
```

3. Start

- Apache
- MySQL

from XAMPP.

4. Open phpMyAdmin.

5. Create a database named

```
student_result_management
```

6. Import

```
student_result_management.sql
```

7. Run

```
http://localhost/student_result_management
```
